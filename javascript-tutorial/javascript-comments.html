자바스크립트 주석

자바스크립트 주석(comments)은 스크립트를 설명함으로써 코드를 더 읽기 쉽게 만드는 데 사용된다.

자바스크립트 주석은 실행되지 않으며 다음의 두 가지 형태를 띤다.

  • 한 줄 주석
  • 여러 줄 주석

한 줄 주석

한 줄 주석은 //로 시작해 그 줄의 끝까지 이어진다.

한 줄 주석을 사용하는 예제는 다음과 같다.

// 한 줄 주석을 이용해 아래 코드를 설명한다
var name = "이름";  // 줄 끝에 주석을 집어넣는다

여러 줄 주석

여러 줄 주석은 /로 시작해 /로 끝나고 여러 줄을 담을 수 있다.

여러 줄 주석을 사용하는 예제는 다음과 같다.

/*
    자바스크립트 여러 줄 주석 예제:
    첫 번째 줄...
    두 번째 줄... 
*/
document.write("itinpractice.com에 오신 것을 환영합니다");
