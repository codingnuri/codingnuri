while 문 문법

자바스크립트에서 while 문은 반복적인 동작을 수행할 때 사용된다. 문법은 다음과 같다.

while (수식){
    구문
}
  • 우선 수식이 평가되고 수식이 거짓이면 자바스크립트는 다음 구문으로 실행 흐름을 이동하고 참이면 반복문의 본문을 구성하는 구문을 실행한 후 수식을 다시 평가한다.
  • This cycle continues until expression evaluates to false, at which point the while statement ends and JavaScript moves on.
  • 수식이 거짓으로 평가될 때까지 이 주기가 계속되며, 수식이 거짓으로 평가되는 시점에는 while 문이 끝나고 실행 흐름이 다음 구문으로 계속 이어진다.

while 문 예제

while 문의 예는 다음과 같다.

var count = 1;
while (count < 5){
    console.log("count = " + count);
    count++;
}

위 코드를 실행한 결과는 다음과 같다.

count = 1
count = 2
count = 3
count = 4
